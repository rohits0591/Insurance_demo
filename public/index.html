Axis Max Life Insurance — WxCC Demo Backend

Dummy backend powering two Webex Contact Center outbound AI agent use cases: premium payment reminder + policy renewal/cross-sell.

Use case 1 — Premium payment reminder

GET /api/policy/:policyNumber — fetch policy + due amount
GET /api/policy/by-phone?phone=... — look up policyholder by phone (ANI) for dynamic greetings
POST /api/payment/create-link — generate dummy payment link
GET /pay.html?paymentId=... — customer checkout page
POST /api/payment/complete — completes payment, generates receipt
GET /api/receipt/:receiptId — download receipt PDF

Use case 2 — Policy renewal + cross-sell

GET /api/cron/check-renewals — scans for renewals due in 7 days, fires webhook (runs daily via Vercel Cron, also callable manually)
GET /api/campaign?days=7 — pull-based dial list alternative
GET /api/campaign?log=true — audit trail of fired triggers
POST /api/webex — mock endpoint to test webhook delivery
GET /api/webex — see what the mock receiver got

Utilities

POST /api/customers — seed dummy policyholders
GET /api/customers — list all policyholders

See README.md in the project for full setup + curl examples.
